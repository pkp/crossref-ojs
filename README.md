# OJS Crossref Export Plugin

This plugin for OJS 3 generates metadata information for articles and issues for indexing in Crossref.
Details on the XML format and data requirements is available at:
https://www.crossref.org/documentation/schema-library/

## Installation

The plugin is available as a git submodule, and it comes together with OJS.

The export functionality can then be accessed through DOI management page
